
# Mini project: The Lorenz System & Deterministic Chaos
## Explore the Lorenz system and its attractor

The Lorenz equations (1963):
  + dx/dt = σ(y - x)
  + dy/dt = x(ρ - z) - y
  + dz/dt = xy - βz
 
Originally derived from a simplified atmospheric convection model,
they became the founding example of chaos theory.
 
This script has three parts:
1. The butterfly attractor
2. Sensitive dependence on initial conditions (the chaos demonstration)
3. Bifurcation: how the attractor changes as ρ varies
